# Jupyter Notebook on Debian (sid)

This Dockerfile is to build the environment with the followings:

* Debian (sid)
* Python 3.6.0
* Jupyter Notebook
  * numpy
  * scipy
  * scikit-learn
  * matplotlib
  * pandas
  * Pillow
